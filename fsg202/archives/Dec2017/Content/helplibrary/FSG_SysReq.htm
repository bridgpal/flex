System Requirements

Flexera Service Gateway 2

The following table lists the recommended software and hardware configuration for a machine running Flexera Service Gateway 2:

Flexera Service Gateway 2 System Requirements

Requirement

Description

Operating System

Windows 2008 R2 Server, Windows 2012 Server, or Windows 2016 Server

Processor

2 GHz Multicore processor

RAM

8 GB

Microsoft .NET Framework

4.6.1

Disk Space

250 MB

Ports to Open in Firewall

Clients access Flexera Service Gateway 2 on ports 9443 and 8280 by default.

User Account Permissions

User running the Flexera Service Gateway 2 installer needs local Admin permissions.

Protocol

Flexera Service Gateway 2 uses a Windows Communication Foundation (WCF) service, and by default it runs on https protocol.
